# Full Stack Open - Submissions

This repository contains my solutions and submissions for the **Full Stack Open** course by the University of Helsinki

This course focuses on modern web development using:

- JavaScript (ES6+) & TypeScript
- React & Redux
- Node.js & Express
- GraphQL
- MongoDB
- CI / CD w/ GitHub Actions
- Containers
- PostgreSQL

**Each part has it's own directory!**

## Disclaimer

This repository is intended for course submissions only.  
Solutions follow the requirements of the Full Stack Open exercises and are not intended as production-ready applications.
